# Simulação de Epidemia utilizando Autômatos Celulares

Projeto desenvolvido para a disciplina de **Autômatos Celulares**, com o objetivo de simular a propagação de uma doença infecciosa por meio de um autômato celular bidimensional.

## Objetivo

Demonstrar como regras locais simples podem gerar comportamentos complexos, simulando a disseminação de uma epidemia utilizando um modelo baseado em autômatos celulares.

## Tecnologias utilizadas

- Python 3
- NumPy
- Matplotlib


## Regras

- Um indivíduo suscetível pode ser infectado caso possua pelo menos um vizinho infectado.
- A transmissão ocorre conforme uma probabilidade configurável.
- Após determinado número de iterações, um indivíduo infectado recupera-se ou morre.
- Todas as células são atualizadas simultaneamente, caracterizando um autômato celular.
